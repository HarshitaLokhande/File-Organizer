# 📁 Smart File Organizer

A Python desktop GUI application that automatically organizes files into categorized folders and supports undo functionality. Built using Tkinter, this project helps users clean and manage cluttered directories efficiently with a simple interface.


## 🚀 Features

- 📂 Automatically organizes files into categories:
  - Images
  - Documents
  - Videos
  - Audio
  - Code
  - Archives
  - Spreadsheets
  - Presentations
  - Applications
  - Others

- ↩ Undo last organization (restores files to original locations)
- 📊 Generates a detailed report of organized files
- 🖥 Simple and clean GUI using Tkinter


## 🛠 Tech Stack

- Python 3
- Tkinter (GUI)
- OS module
- Shutil module
- Datetime module


## ⚙ How It Works

- User selects a folder through GUI
- Program scans all files in the folder
- Files are sorted into category-based folders
- A log file is created for undo functionality
- A report is generated showing file distribution
